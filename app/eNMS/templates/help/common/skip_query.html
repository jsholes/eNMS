The skip query parameter is a python expression that skips the service if the boolean result is True. Skip is evaluated on a per device basis.

This expression is evaluated to determine what to SKIP. If the expression identifies what to RUN, then negate the result.

If a service is skipped, the Skip Value determines the workflow result (success = True, False). The default value is True (success on skip).

Variable substitution can be performed in this block.
